Welcome! Here you can see my learning journey from June to March, including what I learned each month and the challenges I faced along the way.

June – URL / World Wide Web

In June, I learned about the World Wide Web and how websites are connected through the internet. I also discovered how URLs help people access specific webpages.

Challenge: At first it was difficult to understand how websites are connected and how URLs work.

July – Web Browsers & Searching

In July, I practiced using different web browsers and search engines to find information online. This helped me understand how to navigate the web efficiently.

Challenge: Learning to identify trustworthy sources and effective search strategies was tricky.

August – HTML Tags & Links

In August, I studied HTML tags and links. I learned how tags build the structure of a webpage and how links connect one page to another.

Challenge: Remembering different HTML tags and how to use them correctly was sometimes confusing.

September – HTML Tables

In September, I learned how to create tables in HTML to organize information into rows and columns.

Challenge: Creating tables with correct rows and columns required careful attention to the code.

October – Login Box & Input Types

In October, I practiced creating a login box and different input types used in forms. This helped me understand how websites collect user information.

Challenge: I sometimes made mistakes with the input types and needed to fix the code.

November – Website Planning

In November, I learned how to plan a website before coding. Planning helped me understand how to organize the layout and content.

Challenge: Designing the layout and deciding where to place content required creativity.

December – Multimedia

In December, I explored multimedia elements such as images, audio, and videos to make websites more interactive.

Challenge: Adding multimedia sometimes required adjusting the size and placement correctly.

January – Form Elements

In January, I learned about form elements like text boxes, buttons, and checkboxes that allow users to interact with a webpage.

Challenge: Understanding how forms work and organizing the inputs properly was a challenge.

February – CSS Basics

In February, I studied the basics of CSS and learned how to style webpages using colors, spacing, and layouts.

Challenge: Learning how CSS works with HTML took practice and patience.

March – CSS Font & Color

In March, I learned how to apply fonts and colors using CSS to make webpages look more attractive and visually appealing.

Challenge: Choosing the right colors and fonts while keeping the design clean was sometimes difficult.
